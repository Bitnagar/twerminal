creating a tweet:
- use tweet command
- write tweet <your tweet text>. 

Delete a tweet:
- use --delete or -d flag with tweet command
- Deleting a tweet requires a tweet ID.
- Write tweet --delete <tweet ID>

how to create a poll:
- use tweet command with --poll or -p flag.
- Poll needs 3 arguments to be successfully created = optionCount, Duration and Heading.
- write tweet --poll <optionsCount> <duration> <Poll heading in double quotes>
- then enter options.

How to create a thread:
- use tweet command with --thread or -th flag.
- write tweet -th to make a thread.
- then enter tweets as many as you like.

Search tweets related to any topics
- use tweet -srch or --search <topic name> to search any topic of your choice.

Get most recent tweets of your home timeline
- use tweet -tl or --thread option to get the most recent tweets of your timeline.


How to use special characters in tweets (try escaping any special character):
-----in powershell:
- doubles quotes are escaped with backslash  \  and backtick   ` ` .
- single qoutes, @ symbol are escaped with only backticks ` ` .
